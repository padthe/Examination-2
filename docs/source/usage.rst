Usage
=====

.. _installation:

Installation
------------

To use Examination-2, install it using Git Clone.

.. code-block:: console

.. autofunction::
   (.venv) $ git clone git@github.com:padthe/Examination-2.git


Starting the Game
----------------

Use the cd (Change directory) command to navigate to the directory where your Python script is located.
Folder called ``cd Examination-2``:

.. autofunction:: cd Examination-2

To Run the python script use "python main.py" to start the game. 

.. autoexception:: python main.py

For example:
>>> git clone git@github.com:padthe/Examination-2.git
>>> cd Examination-2/src/bj
>>> python main.py
